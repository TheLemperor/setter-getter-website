Object Oriented Programming in Java

Unlike C++, Java does not have header files XXXXXX. Instead, Java is much more class oriented. This means that all files, including the main file, are essentially classes

In Java, every class contains attributes and methods. Attributes are properties that every object of the class contains. And like C++, these attribures can be set to public or private, or even constant ()
